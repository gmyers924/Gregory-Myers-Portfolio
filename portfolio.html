Universal Mental Health (Global Awarness Project)
  • Date Created: October 20th, 2024
  • Technologies Used: Amazon Polly (Text-Speech), Bootstrap, Laravel, CSS3, HTML5, PHP, XAMPP, JavaScript, Visual Studio, Google Maps (Embedded Map), Microsoft Word
  • Skills Demonstrated: Web Development, Community Integration, Audio/Video Implementation, JavaScript Interactivity, Bootstrap Design, Backend Development, Google Maps Customization

Development Details: For this project, I aimed to create a comprehensive website focused on mental health awareness and community engagement. Using Laravel as the foundation, I incorporated multiple features designed to improve functionality and user experience. I developed interactive JavaScript buttons, such as a dynamic toggle icon for the audio text-to-speech feature, and built a multi-page structure with breadcrumbs, dropdown navigation, and a hero image. Leveraging Bootstrap, I implemented an accordion-style FAQ section to provide answers to common mental health questions.
Additionally, I created a contact form that confirms successful submissions, integrated a login and registration system for the community page, and added embedded video and audio content. I also designed a Google Map with custom data relevant to mental health resources, which was embedded into the site.
Some challenges I faced included organizing all the files correctly into directories, troubleshooting the JavaScript functionality for the audio toggle button, and mastering the Bootstrap accordion menu. However, I stayed persistent, researched extensively, and tested thoroughly until these components were functioning seamlessly.

Description: UniversalMentalHealth.org is a dual-purpose site offering mental health information and a community space for interaction. The website provides resources for various mental health topics, a platform for users to connect, and tools to break the stigma surrounding mental health. From information on mental health concerns to an interactive community page, our site ensures users have the tools they need to thrive.

Reason for Selection: This project highlights my ability to integrate multiple technologies into a cohesive, interactive website. It demonstrates my skills in development, design, and user engagement, and reflects the growth of my technical expertise. I’m especially proud of its accessibility features, custom Google Map integration, and community-driven functionality, which align with my passion for impactful web design.

Result: The result is a highly interactive and informative website that supports users with a wealth of mental health resources and fosters a vibrant online community. Its design and functionality continue to evolve, ensuring it remains a valuable resource for users worldwide.
